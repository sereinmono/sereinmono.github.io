## 为什么这个博客长成了这个样子？

现代 UI/UX 设计理念随着技术的发展不断进化。在个人电脑发展的早期，CRT 屏幕能显示的像素极其有限，这极大地限制了电脑屏幕显示的精度，于是乎第一代图形化 UI 诞生时，它便是 **像素风格** 。它是一个计算机“筚路蓝缕，以启山田”时代的象征。

在人们开始熟练操控移动终端，且 AR/VR 技术方兴未艾之时，拟态设计、Liquid Glass 等新的设计语言开始出现。——但是稍等一下，这带给我们一个问题：在一切场景下，人们都需要十分精致、华丽的 UI 设计吗？

> 克制是一种美德。在设计中，克制的风格往往能传达更深层的意义。

于一个 Blog 界面而言，它最应该做的便是能够优雅、简洁地传达作者希望传达的信息。在设计这个 Blog 时，我采取了略显怀旧的，“回头看”的态度，将像素设计语言和 **Neo-Brutalism** （新粗野主义） 相结合，并采取了类似于纸质印刷物的排版布局，来将直观的设计形式和人文的温暖相结合。毕竟，在这个 AI 即将颠覆我们与计算机的交互方式的时刻，在一个从图形语言到自然语言新旧时代的交界点去回望上一个时代的交界点，总会带给我们一些历史的幽默感。

## 这种配色是怎么来的？

这种配色借鉴了任天堂的红白机。它是一个特殊年代的象征：在80年代，人们的计算机制造技术还不能够制造出足够轻薄、便携、现代化的个人计算机，而就是在这样的技术背景下，任天堂利用以娱乐为中心的计算机打开了一条“电脑进万家”的新路。它仍然具有上个世纪个人电脑典型的冷峻外形，但在配色上，白色 + 红色的配色令人眼前一亮，使得它的亲和力因为这样的设计而得到提高。它是一个人文时代精神的象征，代表着计算机可以更加向“人”的存在靠近。

```css
/* 核心配色 */
--color-bg-primary: #FAF8F5;
--color-accent: #8B1A1A;
--color-border: #1A1A1A;
```

## 所以为什么是衬线字体？

衬线字体是一个纸质印刷时代的象征，代表着人文精神，代表着温暖与可靠。在我们与计算机的交流和接触随着 LLM 技术的发展变得越来越多的今天，我们更加需要呼唤这种宝贵的精神——“活人感”。衬线字体温暖而庄重，能够带给我们更多的人文气息。
